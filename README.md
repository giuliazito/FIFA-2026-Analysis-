# FIFA World Cup 2026 Player Performance Analysis

A data analysis and machine learning project exploring player statistics
from the FIFA World Cup 2026 Player Performance Dataset.

## Project Goals
- Exploratory data analysis of player performance metrics
- Data visualization of player/team trends
- Predictive modeling (player rating prediction, clustering, scouting tool)

## Setup
1. Clone this repo
2. Create a virtual environment: `python -m venv venv`
3. Activate it and run: `pip install -r requirements.txt`
4. Download the dataset and place it in `data/raw/` (see Data section below)

## Data
This project uses the FIFA World Cup 2026 Player Performance Dataset.
Dataset not included in this repo — download it separately and place
the CSV in `data/raw/`.
